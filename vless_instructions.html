Инструкция для Windows

Настройка VLESS на компьютере с Windows 10/11

rocket_launch

Happ

Рекомендуемый клиент для Windows

download Скачать Happ
1

Скачайте и установите Happ

Скачайте установщик по кнопке выше и запустите его. Следуйте инструкциям установщика.

2

Запустите программу

После установки запустите Happ. Программа появится в системном трее.

3

Добавьте конфигурацию

Скопируйте вашу VLESS-ссылку (начинается с vless://). Приложение автоматически обнаружит ссылку в буфере обмена и предложит добавить сервер.

4

Установите кастомный роутинг RoscomVPN

Откройте ссылку ниже прямо на устройстве — роутинг автоматически импортируется в Happ:

route Установить роутинг RoscomVPN
5

Подключитесь

Выберите сервер и нажмите кнопку подключения. Готово!

lightbulb

Роутинг RoscomVPN — кастомные правила маршрутизации: заблокированные сайты через VPN, остальной трафик напрямую. Подробнее: GitHub

android

Инструкция для Android

Настройка VLESS на смартфоне Android

apps

Happ

Рекомендуемый клиент для Android

shop Google Play android Скачать APK
1

Установите приложение

Установите Happ из Google Play или скачайте APK-файл напрямую.

2

Добавьте конфигурацию

Способ 1: Скопируйте VLESS-ссылку — приложение автоматически предложит добавить сервер.

Способ 2: Отсканируйте QR-код через меню приложения.

3

Установите кастомный роутинг RoscomVPN

Откройте ссылку ниже прямо на телефоне — роутинг автоматически импортируется в Happ:

route Установить роутинг RoscomVPN
4

Подключитесь

Выберите сервер и нажмите большую кнопку подключения. При первом запуске разрешите создание VPN-соединения.

lightbulb

Роутинг RoscomVPN — заблокированные сайты через VPN, российские ресурсы напрямую. Блокировка рекламы на уровне DNS. Подробнее
